CHIP CONFIGURATION

After having the chip installed to start the ps2 would come out the matrix logo and pressing a series of combinations you could access the matrix menu a bios style menu where you could change the boot of the console to the USB port and put the programs on the usb in this case would require the UlaunchELF that comes to be a menu where you can run programs. ELF programs like freemcboot ,to install it inside the memory card and that allows you to change the system menu ,making as shortcuts to programs already installed or in USB like the UlaunchElf or also the Openps2loader where you can load your copies of ps2 games.
